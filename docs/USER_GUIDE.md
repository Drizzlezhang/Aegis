# Aegis-Trader 用户指南

**交易决策辅助系统** — 帮助你做出更明智的美股交易决策。

> ⚠️ 本系统是**决策辅助工具**，永不自动下单。所有建议仅供参考，真实交易由用户在券商 App 手动完成。

## 功能概览

### 阶段分析
系统基于 Wyckoff 方法论，自动识别标的当前所处的吸筹/派发阶段，并给出阶段转换预警。

### 信号监控
整合 Polymarket 概率、X 社交媒体情绪、宏观新闻等多源信号，实时推送到 Telegram。

### 决策建议
综合阶段分析 + 信号融合 + 虚拟持仓，给出观察/操作建议（仅建议，不自动执行）。

### 回测
使用历史数据回放，验证策略在不同市场环境下的表现。

## 快速开始

1. 配置 `.env` 文件（API keys 等）
2. 启动后端：`python -m src.api.main`
3. 启动前端：`cd web && npm run dev`
4. 访问 `http://localhost:3000`

## 系统边界

- 仅支持美股
- 单用户私有部署
- 桌面 Web 为主，移动端通过 Telegram 接收推送
